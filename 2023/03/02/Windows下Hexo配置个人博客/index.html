Windows下Hexo配置个人博客

发表于2023-03-02|更新于2023-03-02
|阅读量:

Git安装

  • 官网地址，下载并安装
  • 安装完成后以管理员权限打开Git CMD

安装Node.js

  • 官网地址
  • 安装时一路默认直接next到底
  • 安装完成，在windows终端中输入node -v查看是否安装成功

Github仓库

  • 默认已有GitHub账号，新建一个仓库
  • 仓库名称必须为：你的GitHub用户名.github.io
  • 设置仓库类型为公开类型
  • 其余默认，什么都不添加，完成仓库的创建

安装Hexo

  • 在之前打开的Git CMD中输入npm install -g hexo-cli
  • 然后输入hexo -v查看是否安装成功
  • 若提示找不到命令，则找到你hexo的安装地址，如我的是D:\nodejs\node_global，将hexo.cmd所在文件夹添加到系统环境变量中

    也有人将D:\nodejs\node_global\node_modules\hexo-cli\bin这个目录添加到环境变量中，我没有遇到，具体视自己的情况而定

  • 如果嫌慢，可以换源，这里省略

配置Git

  • 在git cmd中输入以下内容：
  • 输入git config -l查看是否配置成功
  • 创建秘钥连接github：
    • 输入ssh-keygen -t rsa -C "你的邮箱"创建秘钥
    • 打开终端提示的秘钥保存地址
    • 打开其中的以.pub结尾的文件，复制其中的值
    • 打开github，进入个人头像中的settings界面
    • 左边找到ssh配置界面，选择新建，随意取名并将刚才复制的内容粘贴进下面的框中
    • 输入ssh -T git@github.com来测试连接

初始化Hexo

  • 新建一个文件夹用来存放你的个人博客文件
  • git cmd跳转到该文件夹下，执行hexo init 项目名称
  • git cmd跳转到刚才建立的文件夹，输入npm i
  • 至此初始化完成，输入hexo s就可在本地4000端口查看个人博客

挂载到github，让其他人都能访问

  • 在git cmd中输入npm install hexo-deployer-git --save

  • git cmd中输入hexo clean && hexo g && hexo d
  • 在浏览器中任何人输入你的用户名.github.io就可以访问了
